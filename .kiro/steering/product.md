# Product Overview

LoomGraph is a fast, form-based webinar banner generator that creates professional LinkedIn-optimized banners (1200x627px) in under 60 seconds.

## Core Value Proposition
- No design skills required - simple form interface
- Real-time preview with instant updates
- Professional templates optimized for LinkedIn
- High-quality export (2x resolution PNG)
- Brand customization with accent colors

## Target Users
Founders and professionals who need quick, professional webinar banners without using complex design tools.

## Current Phase
**Phase 1A MVP** - Single speaker, minimalist template, core functionality complete

## Key Features
- Webinar details form (title, speaker, headshot)
- Date/time with timezone support
- Brand color customization
- Real-time canvas preview
- High-quality PNG export

## Design Philosophy
"Soft Depth" Bento aesthetic:
- Neutral gray backgrounds (#F9FAFB)
- Clean card-based sections
- Subtle shadows and 1px borders
- High-contrast action colors (Electric Blue #3B82F6)

## Security Posture
Enterprise-grade security with CSP, input sanitization, file validation, and secure headers.
